# Currículo
 
 Este diretório mantém versões do currículo como fontes versionadas e PDFs prontos:
 - JSON Resume (PT/EN): `./json/`
 - LaTeX (PT/EN): `./latex/`
 - PDFs: `./pdf/`
 - Carta de apresentação (PT/EN): `./cover-letter/`
 
 O fluxo recomendado para candidaturas é **enviar um PDF** e manter o conteúdo fonte (JSON/LaTeX) versionado para atualizações rápidas.
 
 ## Exportação (opcional)
 Caso você queira gerar uma versão em HTML/PDF a partir do JSON, use um renderer compatível do ecossistema JSON Resume (por exemplo, `resume-cli` + um tema).
