# JSON Resume
 
 Este diretório mantém o currículo no padrão **JSON Resume** como fonte versionada (PT/EN):
 - `resume.pt.json`
 - `resume.en.json`
 
 O fluxo recomendado para candidaturas é **enviar um PDF** (por exemplo, a versão em LaTeX) e usar este JSON para manutenção/atualização rápida.
 
 ## Exportação (opcional)
 Caso você queira gerar uma versão em HTML/PDF a partir do JSON, use um renderer compatível do ecossistema JSON Resume (por exemplo, `resume-cli` + um tema).
